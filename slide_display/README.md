# Slide Display

Минимальный веб-экран для показа библейской цитаты на большом экране.

Открыть:

```bash
python tools/slide_server.py --host 0.0.0.0 --port 8765
```

Затем открыть в браузере:

```text
http://127.0.0.1:8765
```

Страница автоматически слушает live-события от `tools/slide_server.py`.
Когда `tools/vosk_grammar_probe.py` распознает новую ссылку, сервер отправит на
страницу новый код цитаты и текст из `rst.json`, а старая цитата заменится.

Можно также редактировать код цитаты и текст справа вручную, затем нажать
`Показать`. Кнопка `На весь экран` скрывает панель управления и разворачивает
слайд.

Одной командой можно запустить live-распознавание и сервер слайда:

```bash
.venv/bin/python tools/vosk_grammar_probe.py --require-approval --slide-output both
```

Горячие клавиши:

- `Ctrl+Enter` - применить введенный текст.
- `Esc` - выйти из режима презентации.
